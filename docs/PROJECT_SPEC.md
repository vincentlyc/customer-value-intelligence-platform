# Project Spec

## Project name

Customer Value Intelligence Platform

## Objective

Build a demo-ready analytics project that looks like a realistic business architecture / solution architecture portfolio piece.

## Primary audience

- hiring managers
- business stakeholders
- solution architects
- data platform leads
- non-technical executives

## Business problems to simulate

- Customer lifetime value concentration
- Revenue distribution across stores and channels
- Churn-risk customer identification
- Campaign response analysis
- Segment-level decision support

## Required deliverables

- reproducible synthetic datasets
- modular transformation pipeline
- business-readable README
- charts suitable for presentation
- lightweight interactive demo app
- acceptance criteria for Codex execution

## Design principles

- clear separation of data generation / transformation / visualization
- architecture-first storytelling
- simple local execution
- realistic enough for interviews and demos
- extendable by Codex without notebook dependency
